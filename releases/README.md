# PrivyMesh Releases

This repository contains release binaries for PrivyMesh.

## Latest Release

Check latest.txt for the latest version number, or browse the versioned directories.

## Installation

Download the appropriate binary for your platform from the latest release directory.

### Windows
- Extract privymesh-{version}-windows-amd64.zip
- Run privymeshd.exe install to install the daemon
- Use privymesh.exe to interact with the daemon

### Linux
- Download privymeshd-linux-{arch} and privymesh-linux-{arch}
- Make executable: chmod +x privymeshd-linux-* privymesh-linux-*
- Install daemon: sudo ./privymeshd-linux-{arch} install
- Use CLI: ./privymesh-linux-{arch} up

For more information, visit the main repository.
